# rise-local-lead-creation

Complete Agent Orchestration System:

*   Zapier AI agents for workflow orchestration
*   Claude agents for intelligent decision-making
*   MCP server as unified tool gateway
*   Hybrid data architecture (Supabase + Zapier Tables)

This system is designed for efficient lead creation through the orchestration of multiple AI agents. Zapier agents handle workflow automation, while Claude agents provide intelligent decision-making capabilities. The MCP server acts as a central hub, and the hybrid data architecture ensures robust and flexible data management.
